## Introducción
Este proyecto demuestra el uso de Git y GitHub para control de versiones en un script de preprocesamiento de datos con Pandas.

## Comandos Git utilizados
```
| Comando | Propósito |

| git clone | Clonar repositorio remoto |
| git config | Configurar identidad |
| git add | Agregar cambios al staging |
| git commit | Guardar cambios localmente |
| git push | Subir cambios a GitHub |
| git checkout -b | Crear y cambiar a nueva rama |
| git branch | Listar ramas |
| git pull | Traer cambios del remoto |
```

## Automatización con GitHub Actions
Se creó un workflow en `.github/workflows/ci.yml` que se activa con cada push o pull request, ejecutando pruebas y validando el código Python.
